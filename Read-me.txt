Walmart Data Warehouse Project 
Complete Setup Guide


Prerequisites:
1. Python 3.8+ installed
2. MySQL Server installed
3. Required Python packages: pandas, mysql-connector-python

Step 1: Database Setup

1. Run the database setup script:
   python setup_database.py
   
2. Enter your MySQL root password when prompted
   
3. This will:
   - Create 'walmart_dw' database
   - Create all dimension and fact tables
   - Load customer and product master data

Step 2: Run HYBRIDJOIN ETL Process
1. Execute the HYBRIDJOIN algorithm:
   python Hybrid-Join.py
   
2. Enter your database credentials when prompted:
   - Host (usually 'localhost')
   - Username (usually 'root') 
   - Password (your MySQL password)
   - Database name ('walmart_dw')

3. The program will:
   - Load master data into memory
   - Process transactional data using HYBRIDJOIN algorithm
   - Enrich transactions with customer and product data
   - Load results into the fact table
   - Show progress every 10 seconds

Step 3: Execute OLAP Queries
1. Open MySQL Workbench or your preferred SQL client
2. Connect to the 'walmart_dw' database
3. Run the queries from 'Queries-DW.sql' file
4. Analyze the results for business insights

File Structure:
- Create-DW.sql: Database schema creation script
- setup_database.py: Database initialization and master data loading
- Hybrid-Join.py: Main HYBRIDJOIN algorithm implementation
- Queries-DW.sql: 20 OLAP analysis queries
- customer_master_data.csv: Customer demographic data
- product_master_data.csv: Product and store information  
- transactional_data.csv: Sales transaction data

Expected Output:
- Database with star schema populated
- Fact table (FactSales) with enriched transaction data
- Ability to run all 20 OLAP queries for business intelligence
- Near-real-time data processing simulation

Troubleshooting:
1. Ensure MySQL server is running
2. Check database credentials are correct
3. Verify all CSV files are in the same directory
4. Make sure Python packages are installed:
   pip install pandas mysql-connector-python

Note: The HYBRIDJOIN algorithm simulates near-real-time processing by reading the CSV file and processing transactions with small delays to mimic streaming data.
